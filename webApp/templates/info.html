About SNAPS

SNAPS (Simple NMR Assignments from Predicted Shifts) is a program for assigning NMR spectra using information from chemical shift predictions derived from protein structures. It is currently under development.

To use SNAPS, you will need:

  • a chemical shift list derived from one or more protein spectra. The shifts should be grouped into spin systems/pseudo-residues. A number of formats are accepted, including SPARKY, XEASY, nmrPipe and shift lists exported from CCPN Analysis (v2.x). In future, it will be possible to import peaklists directly.
  • a list of predicted chemical shifts. Currently, SHIFTX2 and SPARTA+ predictions are accepted.
You can also leave both input fields blank, and the server will return results from some demonstration data.

After running SNAPS, two pieces of information are returned:

  • a table containing the maximum likelihood assignment.
  • an interactive strip plot which allows you to quickly locate residues with poor sequential links. (This will only be generated if your input data contains sequential connectivity information for at least one carbon, eg. both CA_i and CA_i-1 shifts.)

SNAPS is available at GitHub: you can find it at https://github.com/aheyam/NAPS. Any feedback would be gratefully received at alex.heyam at york.ac.uk.

How it works

SNAPS works by assigning a probability to each possible pairing between observed spin systems and predicted residues. This is calculated assuming the error in the predicted shift follows a Gaussian distribution, with a standard deviation determined from a set of roughly 60 test proteins. Probabilities for each atom type are multiplied to get the overall spin system-residue match probability. Information on up to 9 atom types can be incorporated (H, HA, N, C, CA and CB, and also C, CA and CB of the preceeding residue).

SNAPS then finds the assignment which maximises the overall probability (ie. the product of the probabilities of the individual peak assignments). This is done efficiently using a linear optimisation algorithm for solving general assignment problems. Note that chemical shift matching between adjacent residues is *not* used when generating the assignments (except indirectly), so this can be used to assess assignment quality. SNAPS generates a strip plot highlighting areas of sequential mismatch which require manual inspection.

Disclaimer

SNAPS is currently under development, and is available for demonstration or testing purposes only. It likely contains bugs, and may change without notice. Use at your own risk!
